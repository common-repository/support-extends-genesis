This plugin allows you adjust some basic components of website that using genesis framework.

== Description ==

Genesis have many actions and filters which allow you change the components of the theme.
If you have no experiences about the hooks API, you may feel so hard to use actions and filters in Genesis.
This small plugin will help you make the changes on genesis site easier.

= Features =
* Header
* Title Bar
* Footer
* Custom post type templates
* Custom style

== Installation ==

You can install the plugin by two ways:

1. Through Wordpress admin panel:

	- Goto "Plugins -> Add new". Then type "wp-nice-search" into search box.
	- Then click "Install Now" button and activate it.
	(if you have already a plugin on zip file, just click "Upload plugin" on the top and choose the plugin zip file.)

2. Upload through FTP

	- Upload plugin folder into plugins directory.
	- Activate it in admin.
